SUEWS_Water.txt
~~~~~~~~~~~~~~~

`SUEWS_Water.txt` specifies the characteristics for the water surface
cover type by linking codes in column 1 of `SUEWS_Water.txt` to the codes
specified in `SUEWS_SiteSelect.txt` (`Code_Water`).
