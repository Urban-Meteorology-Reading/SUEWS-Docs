6.2.2. SUEWS_Conductance.txt¶

SUEWS_Conductance.txt contains the parameters needed for the Jarvis (1976) surface conductance model used in the modelling of evaporation in SUEWS. These values should not be changed independently of each other. The suggested values below have been derived using datasets for Los Angeles and Vancouver (see Järvi et al. (2011) [J11]) and should be used with gsModel=1. An alternative formulation (gsModel=2) uses slightly different functional forms and different coefficients (with different units).
